# 📚 Sistema de Livraria

Sistema de gerenciamento de livraria desenvolvido em **Java**, com foco na prática de conceitos fundamentais de programação orientada a objetos e organização de sistemas.

---

## 📌 Sobre o Projeto

O **Sistema de Livraria** é uma aplicação que simula o funcionamento básico de uma livraria, permitindo o gerenciamento de livros e operações relacionadas.

O projeto foi desenvolvido com o objetivo de consolidar conhecimentos em:

- Programação Orientada a Objetos (POO)
- Estruturação de projetos em Java
- Manipulação de dados em memória
- Organização de código em camadas (básico)

---

## 🚀 Tecnologias Utilizadas

- Java
- Paradigma Orientado a Objetos (POO)

---

## ⚙️ Funcionalidades

- 📖 Cadastro de estudante, livro e aluguel 
- 📋 Listagem de estudante, livro e aluguel 
- 🔍 Busca de estudante, livro e aluguel 
- ✏️ Atualização de informações de de estudante, livro e aluguel
- ❌ Remoção de de estudante, livro e aluguel

## 💻 Como Executar o Projeto

### 🔧 Pré-requisitos

- Java JDK 8 ou superior instalado

Verifique com:
```bash
java -version
```

### ▶️ Executando
- Clone o repositório:
```bash
git clone git@github.com:GSOliveira1/SistemaBiblioteca.git
```
- Acesse a pasta:
```bash
  cd cd SistemaLivraria
```
- Compile os arquivos:
```bash
javac src/**/*.java
```
Execute a aplicação:
```bash
java src.Program
```
